Datenschutzerklärung

Der Schutz deiner persönlichen Daten ist mir wichtig. Diese Website wird als privates, nicht-kommerzielles Projekt betrieben.

Welche Daten werden erfasst?

Wenn du mir Daten per E-Mail oder über ein Formular übermittelst (z.B. Name, Adresse, Telefonnummer, Angaben zum Tier oder Bilder), werden diese ausschließlich zur Erstellung einer PET-ID verwendet.

Wie werden die Daten verarbeitet?

  • Die Daten werden manuell geprüft und verarbeitet
  • Es erfolgt keine automatische Veröffentlichung
  • Es gibt keine Benutzerkonten oder Logins
  • Es werden keine Tracking- oder Analyse-Tools eingesetzt

Weitergabe von Daten

Es findet keine Weitergabe der Daten an Dritte statt.

Speicherung

Die Daten werden nur so lange gespeichert, wie es für den Zweck der PET-ID erforderlich ist.

Deine Rechte

Du kannst jederzeit Auskunft, Berichtigung oder Löschung deiner Daten verlangen.
